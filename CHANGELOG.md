# Change Log

All notable changes to the "CodeChronicle" extension will be documented in this file.

## [0.0.1] - 2026-03-06

### Initial Release

#### Features
- 📜 **Interactive Timeline**: Complete commit history with expandable diffs
- 🤖 **AI-Powered Insights**: Automatic risk detection and smart recommendations
- 👥 **Author Analytics**: Visual contribution charts showing who wrote what
- 🔗 **Related Files Discovery**: Find files that change together
- 📊 **Code Metrics**: Age, complexity, change frequency analysis
- ⚡ **Performance**: Optimized Git operations with progress indicators
- 🎨 **Beautiful UI**: Native VS Code theme integration

#### Commands
- `Show CodeChronicle`: Right-click on selected code to view its complete history

#### Requirements
- Git repository
- VS Code 1.80.0 or higher
